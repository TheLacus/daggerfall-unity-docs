Class MobilePersonAsset

The component that handles graphics for a mobile person.

Remarks

Implementations should be added to a prefab named "MobilePersonAsset" bundled with a mod. This prefab acts as a provider for all custom npcs graphics. This allows mods to replace classic MobilePersonBillboard with 3d models or other alternatives. The actual graphic asset for specific npc should be loaded when requested by SetPerson(Races, Genders, Int32, Boolean).
